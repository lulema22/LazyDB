# LazyDB

Учебный проект простой базы данных в оперативной памяти на C++.

Проект реализует загрузку данных из CSV-файлов, хранение таблиц в памяти, проверку ограничений целостности и ускоренный поиск с помощью индексов (HashIndex и B-Tree).

---

## 📌 Основные возможности

- Загрузка таблиц из CSV-файлов
- Хранение данных в оперативной памяти
- Поддержка первичного ключа (Primary Key)
- Проверка ограничений целостности данных:
  - UNIQUE
  - FOREIGN KEY
  - RESTRICT при удалении
- Добавление, удаление и обновление записей
- Быстрый поиск данных через индексы:
  - HashIndex — поиск по равенству
  - BTreeIndex — поиск по диапазонам
- Разделение логики хранения, индексации

## 📂 Структура проекта

Проект расположен в **одной ветке** и имеет следующую структуру:

```text
.
├── data/                 # CSV-файлы с данными (база данных)
│   ├── addresses.csv
│   ├── departments.csv
│   ├── employees.csv
│   ├── suppliers.csv
│   ├── products.csv
│   └── purchases.csv
│
├── model/                # Модели данных
│   ├── Address.h / .cpp
│   ├── Department.h / .cpp
│   ├── Employee.h / .cpp
│   ├── Supplier.h / .cpp
│   ├── Product.h / .cpp
│   └── Purchase.h / .cpp
│
├── HashTable.h            # Реализация хеш-таблицы
├── BTree.h                # Реализация B-Tree
├── Index.h                # Интерфейс и реализации индексов
├── Table.h                # Универсальная таблица хранения данных
├── Database.h             # Класс базы данных
├── DbErrors.h             # Ошибки и ограничения целостности
├── gui_main.cpp           # Точка входа / GUI
└── README.md
